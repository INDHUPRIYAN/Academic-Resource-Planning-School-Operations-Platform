Instructions & Template Guide

Please structure your workbook sheets exactly as follows:

1. Classes & Sections

  • Class Name: Name of class/grade (e.g. "Grade 9")
  • Section Name: Letter or name of section (e.g. "A")

2. Resources

  • Resource Name: Unique room/facility name (e.g. "Chemistry Lab")
  • Type: Optional type of resource (e.g. "Lab")
  • Capacity: Optional student capacity (e.g. "30")

3. Subjects

  • Subject Name: Course name (e.g. "Chemistry")
  • Weekly Hours: Standard hour count (e.g. "4")
  • Resource Name: Optional name matching a Resource tab entry

4. Teachers

  • Teacher Name: Display name (e.g. "John Doe")
  • Email: Credentials login (must be unique)
  • Password: Initial login password
  • Department: Optional (e.g. "Science")
  • Max Weekly Hours: Weekly limits (defaults to 30)
  • Subjects: Comma-separated subject names matching the Subjects tab
